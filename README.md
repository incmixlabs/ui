# Incmix Monorepo

## Scaffolded from turborepo-nextjs

## The Tech stack is

### Client

- [x] pnpm
- [x] react
- [x] nextjs
- [x] zustand
- [x] tailwind
- [x] radix-ui and shadcn
- [x] lucia auth

### Server

- [x] pnpm
- [x] hono
- [ ] backend postgres (porsager/postgres)
